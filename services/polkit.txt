package:     polkit
service:     polkitd
description:
  A privilege management service that allows unprivileged processes to
  request elevated permissions in a controlled, policy-based way without
  requiring full sudo access. Examples: allowing a normal user to reboot,
  mount drives, manage NetworkManager connections, or install packages via
  a GUI. Many desktop environments and GUI tools silently depend on polkit.

  Requires dbus and elogind to function correctly on Void.

  After install, the runit service is at /etc/sv/polkitd.
